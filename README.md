# AthkarApp | تطبيق أذكار

[![.NET MAUI](https://img.shields.io/badge/.NET%20MAUI-512BD4?style=for-the-badge&logo=dotnet&logoColor=white)](https://dotnet.microsoft.com/en-us/apps/maui)
[![Open Source](https://img.shields.io/badge/Open%20Source-%E2%9D%A4-red?style=for-the-badge)](https://github.com/omarabdelsalm/AthkarApp)

AthkarApp is a modern, cross-platform application built with **.NET MAUI** designed to provide a comprehensive experience for reading and listening to the Holy Quran and Athkar (daily supplications).

تطبيق **AthkarApp** هو تطبيق حديث متعدد المنصات تم بناؤه باستخدام تقنية **.NET MAUI**، مصمم ليقدم تجربة شاملة لقراءة والاستماع للقرآن الكريم والأذكار اليومية.

---

## ✨ Features | المميزات

- **📖 Quran Explorer:** Browse all Surahs with a clean and intuitive interface.
  **تصفح القرآن:** عرض جميع السور بواجهة مستخدم بسيطة وسهلة.
- **🔊 Smart Recitation:** Listen to the full Surah or individual verses (Ayahs).
  **تلاوة ذكية:** إمكانية الاستماع للسورة كاملة أو لكل آية بشكل منفصل.
- **🎧 High Quality Audio:** Featuring the recitation of Sheikh Mishary Rashid Alafasy.
  **صوت عالي الجودة:** يتضمن تلاوة الشيخ مشاري راشد العفاسي.
- **🌍 Full RTL Support:** Designed specifically for Arabic language users.
  **دعم كامل للغة العربية:** تصميم متوافق مع اتجاه الكتابة من اليمين إلى اليسار (RTL).
- **⚡ Fast & Light:** Optimized performance for a smooth experience on Android, iOS, and Windows.
  **سريع وخفيف:** أداء محسن لتجربة سلسة على مختلف أنظمة التشغيل.

---

## 🛠️ Tech Stack | التقنيات المستخدمة

- **Framework:** [.NET MAUI](https://dotnet.microsoft.com/en-us/apps/maui) (Multi-platform App UI)
- **Language:** C#
- **API:** [AlQuran Cloud API](https://api.alquran.cloud/) for Quran data.
- **Audio Source:** [QuranicAudio](https://quranicaudio.com/) for recitations.
- **Audio Plugin:** [Plugin.Maui.Audio](https://github.com/jfversluis/Plugin.Maui.Audio)

## 🚀 Getting Started | كيف تبدأ

### Prerequisites | المتطلبات
- Visual Studio 2022 (with .NET MAUI workload installed).
- .NET 8.0 or 9.0 SDK.

### Setup | الإعداد
1. Clone the repository | تحميل المشروع:
   ```bash
   git clone https://github.com/omarabdelsalm/AthkarApp.git
   ```
2. Open the solution file `AthkarApp.slnx` in Visual Studio.
3. Restore NuGet packages and run the project on your preferred emulator or device.
